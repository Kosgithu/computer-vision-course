# 5.3 - Détection de Piétons (CCTV)

## 🎯 Concept

La détection de piétons en temps réel à partir d'une caméra de surveillance (CCTV) utilise des classificateurs Haar Cascade spécifiquement entraînés pour reconnaître les silhouettes humaines. Cette technologie est largement utilisée dans les systèmes de surveillance intelligente.

### Points clés
- **Traitement temps réel** : Analyse en direct du flux vidéo
- **Classificateur fullbody** : `haarcascade_fullbody.xml` détecte le corps entier
- **Classificateur upperbody** : `haarcascade_upperbody.xml` détecte le haut du corps (visage + torse)
- **Optimisation CCTV** : Paramètres adaptés aux caméras fixes avec éclairage variable

---

## 💻 Code Minimal

```python
import cv2
import time

# Chargement des classificateurs
body_detect = cv2.CascadeClassifier('haarcascade_fullbody.xml')
upperbody_detect = cv2.CascadeClassifier('haarcascade_upperbody.xml')

# Ouverture de la source vidéo
capture = cv2.VideoCapture(0)  # ou 'video.mp4' pour un fichier

while capture.isOpened():
    ret, frame = capture.read()
    if not ret:
        break
    
    gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
    
    # Détection corps entier
    bodies = body_detect.detectMultiScale(gray, 1.1, 3)
    
    # Détection haut du corps
    upper_bodies = upperbody_detect.detectMultiScale(gray, 1.1, 3)
    
    # Dessin des rectangles
    for (x, y, w, h) in bodies:
        cv2.rectangle(frame, (x, y), (x+w, y+h), (0, 255, 0), 2)
        cv2.putText(frame, 'Personne', (x, y-10), 
                   cv2.FONT_HERSHEY_SIMPLEX, 0.5, (0, 255, 0), 2)
    
    for (x, y, w, h) in upper_bodies:
        cv2.rectangle(frame, (x, y), (x+w, y+h), (255, 0, 0), 2)
        cv2.putText(frame, 'Haut corps', (x, y-10),
                   cv2.FONT_HERSHEY_SIMPLEX, 0.5, (255, 0, 0), 2)
    
    cv2.imshow('Détection Piétons CCTV', frame)
    
    if cv2.waitKey(1) == 27:
        break

capture.release()
cv2.destroyAllWindows()
```

## 🔧 Fonction détaillée

### `cv2.VideoCapture(index)`
```python
capture = cv2.VideoCapture(0)       # Caméra par défaut
capture = cv2.VideoCapture('cctv.mp4')  # Fichier vidéo
```
- **Paramètre** : Index de caméra ou chemin fichier
- **Retour** : Objet VideoCapture

### `capture.isOpened()`
Vérifie si la vidéo/caméra a été ouverte avec succès.
- **Retour** : `True` si ouvert, `False` sinon

### Classificateurs Fullbody vs Upperbody

| Classificateur | Détecte | Utilisation |
|----------------|---------|-------------|
| `fullbody` | Corps entier (tête aux pieds) | Personnes debout, marchant |
| `upperbody` | Haut du corps (tête à la taille) | Personnes assises, partiellement cachées |

### Paramètres de détection
```python
bodies = body_detect.detectMultiScale(gray, 1.1, 3)
```
- **scaleFactor = 1.1** : Scan fin pour détecter différentes tailles
- **minNeighbors = 3** : Équilibré entre précision et faux positifs

## ⚠️ Pièges et Astuces

### ❌ Erreurs fréquentes

**1. Classificateurs non trouvés**
```python
# Les fichiers XML doivent être présents dans le dossier
# Télécharger depuis : https://github.com/opencv/opencv/tree/master/data/haarcascades
```

**2. Confusion fullbody vs upperbody**
- `fullbody` = silhouette complète (mieux pour personnes debout)
- `upperbody` = visage + torse (mieux pour personnes assises/partielles)

**3. Oublier de libérer la caméra**
```python
# ❌ Mauvais
while True:
    ret, frame = capture.read()
    # ...
# Pas de capture.release() → caméra bloquée !

# ✅ Bon
try:
    while True:
        ret, frame = capture.read()
        # ...
finally:
    capture.release()
    cv2.destroyAllWindows()
```

### 💡 Astuces

**Améliorer la détection en conditions difficiles**
```python
# Augmenter minNeighbors pour réduire les faux positifs
bodies = body_detect.detectMultiScale(gray, 1.2, 5)

# Diminuer scaleFactor pour détecter plus de tailles
bodies = body_detect.detectMultiScale(gray, 1.05, 3)
```

**Ajouter un compteur de personnes**
```python
count = len(bodies) + len(upper_bodies)
cv2.putText(frame, f'Personnes: {count}', (10, 30),
           cv2.FONT_HERSHEY_SIMPLEX, 1, (0, 255, 255), 2)
```

**Sauvegarder les détections**
```python
if len(bodies) > 0:
    cv2.imwrite(f'alerte_{time.time()}.jpg', frame)
```

## 🔗 Différences avec la détection de visages

| Aspect | Visages | Piétons |
|--------|---------|---------|
| **Classificateur** | `haarcascade_frontalface` | `haarcascade_fullbody` |
| **Objet détecté** | Visage uniquement | Corps entier ou partie |
| **Mouvement** | Peut être immobile | Souvent en mouvement |
| **Taille** | Relativement constante | Très variable (adulte/enfant) |
| **Contexte** | Frontal de préférence | Toutes orientations |

## 📝 Résumé

La détection de piétons en CCTV combine :
1. **Capture temps réel** avec `VideoCapture`
2. **Détection multi-cascade** (fullbody + upperbody)
3. **Annotation visuelle** avec rectangles et labels
4. **Comptage/Alerte** selon le cas d'usage

**Point clé** : Utiliser deux classificateurs (fullbody + upperbody) permet de détecter les personnes dans plus de situations (debout, assises, partiellement cachées).
