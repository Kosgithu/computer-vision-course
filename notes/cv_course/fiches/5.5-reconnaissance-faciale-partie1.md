# 5.5 - Reconnaissance Faciale - Partie 1

## 🎯 Concept

La reconnaissance faciale va au-delà de la simple détection : elle permet d'**identifier** une personne spécifique parmi plusieurs. Cette première partie couvre la création d'un dataset d'entraînement et l'extraction des caractéristiques faciales.

### Points clés
- **Dataset personnel** : Collection d'images de visages à reconnaître
- **Face ID** : Attribution d'un identifiant numérique à chaque personne
- **Prétraitement** : Détection, rognage et redimensionnement des visages
- **Sauvegarde** : Stockage des images pour l'entraînement (Partie 2)

---

## ⚠️ IMPORTANT : picamera2 vs cv2.VideoCapture

### Pourquoi on utilise picamera2 sur Raspberry Pi 5 ?

Sur **Raspberry Pi 5 avec caméra CSI**, `cv2.VideoCapture(0)` **ne fonctionne PAS** à cause d'un bug avec le backend GStreamer. La solution est d'utiliser **picamera2** pour capturer, puis OpenCV pour le traitement d'image.

### Différences clés

| Aspect | cv2.VideoCapture | picamera2 |
|--------|------------------|-----------|
| **Plateforme** | PC / Webcam USB | Raspberry Pi + CSI |
| **Initialisation** | `cv2.VideoCapture(0)` | `Picamera2()` + `configure()` |
| **Capture** | `capture.read()` | `picam2.capture_array()` |
| **Format couleur** | BGR (OpenCV natif) | RGB (conversion nécessaire) |
| **Arrêt** | `capture.release()` | `picam2.stop()` |

### Conversion obligatoire RGB → BGR

```python
# picamera2 capture en RGB, OpenCV travaille en BGR
frame = picam2.capture_array()           # Format RGB
frame = cv2.cvtColor(frame, cv2.COLOR_RGB2BGR)  # Conversion BGR
```

Sans cette conversion, les couleurs seront inversées (rouge ↔ bleu) !

## 💻 Code Minimal

### Version Standard (PC/Webcam USB)

```python
import cv2
import os

# Création du dossier dataset
face_id = input('ID de la personne : ')
face_name = input('Nom de la personne : ')
os.makedirs(f"dataset/{face_name}", exist_ok=True)

# Classificateur visage
face_cascade = cv2.CascadeClassifier('haarcascade_frontalface_default.xml')

# Capture
capture = cv2.VideoCapture(0)
count = 0

while capture.isOpened() and count < 30:
    ret, frame = capture.read()
    if not ret:
        break
    
    gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
    faces = face_cascade.detectMultiScale(gray, 1.3, 5)
    
    for (x, y, w, h) in faces:
        count += 1
        # Sauvegarde du visage rogné
        face_roi = gray[y:y+h, x:x+w]
        cv2.imwrite(f"dataset/{face_name}/{count}.jpg", face_roi)
        
        # Rectangle de confirmation
        cv2.rectangle(frame, (x, y), (x+w, y+h), (0, 255, 0), 2)
    
    cv2.putText(frame, f'Photos: {count}/30', (10, 30),
               cv2.FONT_HERSHEY_SIMPLEX, 1, (0, 255, 0), 2)
    cv2.imshow('Collecte Dataset', frame)
    
    if cv2.waitKey(100) == 27:
        break

capture.release()
cv2.destroyAllWindows()
print(f"Dataset créé : {count} images")
```

### Version Raspberry Pi 5 + Caméra CSI (picamera2)

```python
import cv2
import os
from picamera2 import Picamera2
import time

# Création du dataset
face_id = input('ID de la personne : ')
face_name = input('Nom de la personne : ')
os.makedirs(f"dataset/{face_name}", exist_ok=True)

face_cascade = cv2.CascadeClassifier('haarcascade_frontalface_default.xml')

# Initialisation picamera2
picam2 = Picamera2()
picam2.configure(picam2.create_preview_configuration(
    main={"format": "RGB888", "size": (640, 480)}
))
picam2.start()
time.sleep(0.1)

count = 0

while count < 30:
    # Capture et conversion
    frame = picam2.capture_array()
    frame = cv2.cvtColor(frame, cv2.COLOR_RGB2BGR)
    
    gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
    faces = face_cascade.detectMultiScale(gray, 1.3, 5)
    
    for (x, y, w, h) in faces:
        count += 1
        face_roi = gray[y:y+h, x:x+w]
        cv2.imwrite(f"dataset/{face_name}/{count}.jpg", face_roi)
        cv2.rectangle(frame, (x, y), (x+w, y+h), (0, 255, 0), 2)
    
    cv2.putText(frame, f'Photos: {count}/30', (10, 30),
               cv2.FONT_HERSHEY_SIMPLEX, 1, (0, 255, 0), 2)
    cv2.imshow('Collecte Dataset', frame)
    
    if cv2.waitKey(100) == 27:
        break

picam2.stop()
cv2.destroyAllWindows()
print(f"Dataset créé : {count} images")
```

## 🔧 Fonction détaillée

### `os.makedirs()`
```python
os.makedirs(f"dataset/{face_name}", exist_ok=True)
```
- Crée les dossiers récursivement
- `exist_ok=True` : pas d'erreur si le dossier existe déjà

### Extraction du ROI (Region of Interest)
```python
face_roi = gray[y:y+h, x:x+w]
```
- Récupère uniquement la zone du visage détecté
- Permet de sauvegarder une image normalisée

### `Picamera2()` (Raspberry Pi)
```python
from picamera2 import Picamera2
picam2 = Picamera2()
picam2.configure(picam2.create_preview_configuration(
    main={"format": "RGB888", "size": (640, 480)}
))
picam2.start()
```

## ⚠️ Pièges et Astuces

### ❌ Erreurs fréquentes

**1. Oublier la conversion RGB→BGR (picamera2)**
```python
# ❌ Mauvais - Haar Cascade échoue sur RGB
frame = picam2.capture_array()
gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)

# ✅ Bon - Conversion d'abord
frame = picam2.capture_array()
frame = cv2.cvtColor(frame, cv2.COLOR_RGB2BGR)
gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
```

**2. Pas assez de variété dans les photos**
```python
# ❌ Mauvais - 30 photos identiques
# La reconnaissance échouera sur de nouveaux angles

# ✅ Bon - Varier angles, expressions, éclairage
# Bouger la tête pendant la capture
```

**3. Dossier non créé**
```python
# ❌ Erreur si le dossier n'existe pas
cv2.imwrite(f"dataset/{name}/1.jpg", face)  # FileNotFoundError

# ✅ Toujours créer les dossiers
os.makedirs(f"dataset/{name}", exist_ok=True)
```

### 💡 Astuces

**Compteur visuel**
```python
cv2.putText(frame, f'Photos: {count}/30', (10, 30),
           cv2.FONT_HERSHEY_SIMPLEX, 1, (0, 255, 0), 2)
```

**Délai entre captures**
```python

```

**Vérification qualité visage**
```python
if w > 100 and h > 100:  # Visage suffisamment grand
```
