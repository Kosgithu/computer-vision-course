# 5.6 - Reconnaissance Faciale - Partie 2

## 🎯 Concept

La Partie 2 de la reconnaissance faciale utilise le dataset créé précédemment pour entraîner un modèle de reconnaissance. OpenCV fournit l'algorithme **LBPH (Local Binary Patterns Histograms)** qui apprend les caractéristiques uniques de chaque visage.

### Points clés
- **LBPH Face Recognizer** : Algorithme de reconnaissance faciale intégré à OpenCV
- **Entraînement** : Apprentissage sur les images du dataset
- **Sauvegarde modèle** : Persistance du modèle entraîné
- **Reconnaissance temps réel** : Identification des visages dans le flux vidéo

---

## ⚠️ IMPORTANT : picamera2 vs cv2.VideoCapture

### Pourquoi on utilise picamera2 sur Raspberry Pi 5 ?

Sur **Raspberry Pi 5 avec caméra CSI**, `cv2.VideoCapture(0)` **ne fonctionne PAS** à cause d'un bug avec le backend GStreamer. La solution est d'utiliser **picamera2** pour capturer, puis OpenCV pour le traitement d'image.

### Différences clés

| Aspect | cv2.VideoCapture | picamera2 |
|--------|------------------|-----------|
| **Plateforme** | PC / Webcam USB | Raspberry Pi + CSI |
| **Initialisation** | `cv2.VideoCapture(0)` | `Picamera2()` + `configure()` |
| **Capture** | `capture.read()` | `picam2.capture_array()` |
| **Format couleur** | BGR (OpenCV natif) | RGB (conversion nécessaire) |
| **Arrêt** | `capture.release()` | `picam2.stop()` |

### Conversion obligatoire RGB → BGR

```python
# picamera2 capture en RGB, OpenCV travaille en BGR
frame = picam2.capture_array()           # Format RGB
frame = cv2.cvtColor(frame, cv2.COLOR_RGB2BGR)  # Conversion BGR
```

Sans cette conversion, les couleurs seront inversées (rouge ↔ bleu) !

## 💻 Code Minimal

### Version Standard (PC/Webcam USB)

```python
import cv2
import os
import numpy as np

# Étape 1 : Entraînement
recognizer = cv2.face.LBPHFaceRecognizer_create()
face_cascade = cv2.CascadeClassifier('haarcascade_frontalface_default.xml')

faces = []
ids = []
names = {}  # id -> nom

# Chargement du dataset
dataset_path = "dataset"
for person_name in os.listdir(dataset_path):
    person_path = os.path.join(dataset_path, person_name)
    person_id = int(input(f"ID pour {person_name} : "))
    names[person_id] = person_name
    
    for img_name in os.listdir(person_path):
        img_path = os.path.join(person_path, img_name)
        face_img = cv2.imread(img_path, 0)
        faces.append(face_img)
        ids.append(person_id)

# Entraînement
recognizer.train(faces, np.array(ids))
recognizer.save('trainer.yml')
print("Modèle entraîné et sauvegardé")

# Étape 2 : Reconnaissance temps réel
capture = cv2.VideoCapture(0)

while capture.isOpened():
    ret, frame = capture.read()
    if not ret:
        break
    
    gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
    faces = face_cascade.detectMultiScale(gray, 1.3, 5)
    
    for (x, y, w, h) in faces:
        face_roi = gray[y:y+h, x:x+w]
        
        # Prédiction
        id_pred, confidence = recognizer.predict(face_roi)
        
        if confidence < 100:  # Plus c'est bas, mieux c'est
            name = names.get(id_pred, "Inconnu")
            color = (0, 255, 0)
        else:
            name = "Inconnu"
            color = (0, 0, 255)
        
        cv2.rectangle(frame, (x, y), (x+w, y+h), color, 2)
        cv2.putText(frame, f'{name} ({confidence:.0f})', (x, y-10),
                   cv2.FONT_HERSHEY_SIMPLEX, 0.8, color, 2)
    
    cv2.imshow('Reconnaissance Faciale', frame)
    if cv2.waitKey(1) == 27:
        break

capture.release()
cv2.destroyAllWindows()
```

```python
import os
import numpy as np

import time

recognizer = cv2.face.LBPHFaceRecognizer_create()
face_cascade = cv2.CascadeClassifier('haarcascade_frontalface_default.xml')

        if confidence < 100:

    if cv2.waitKey(1) == 27:
        break

picam2.stop()
cv2.destroyAllWindows()
```

## 🔧 Fonction détaillée

### `cv2.face.LBPHFaceRecognizer_create()`
```python
recognizer = cv2.face.LBPHFaceRecognizer_create()
```
- Crée un reconnaisseur basé sur LBPH
- **LBPH** = Local Binary Patterns Histograms
- Robuste aux variations d'éclairage

### `recognizer.train()`
```python
recognizer.train(faces, np.array(ids))
```
- **faces** : Liste d'images (tableaux NumPy)
- **ids** : Tableau NumPy des IDs correspondants

### `recognizer.predict()`
```python
id_pred, confidence = recognizer.predict(face_roi)
```
- **Retour** : (ID prédit, niveau de confiance)
- **confidence** : Plus c'est bas, plus c'est sûr (< 100 = bon)

### `Picamera2()` (Raspberry Pi)
```python
from picamera2 import Picamera2
picam2 = Picamera2()
picam2.configure(picam2.create_preview_configuration(
    main={"format": "RGB888", "size": (640, 480)}
))
picam2.start()
```

## 🎨 Analogie Simple

Imagine un portier dans un club VIP :
- Il a un **album photos** de tous les membres (dataset)
- Il **apprend** à reconnaître chaque membre (entraînement)
- Quelqu'un arrive → il **compare** avec ses souvenirs (prédiction)
- Il donne un **score de certitude** (confidence)
- Il laisse entrer si le score est bon, refuse sinon

LBPH fonctionne pareil :
- Apprentissage sur les photos
- Comparaison des patterns locaux
- Score de confiance pour chaque prédiction

## ⚠️ Pièges et Astuces

### ❌ Erreurs fréquentes

**1. Oublier la conversion RGB→BGR (picamera2)**
```python
# ❌ Mauvais - reconnaissance échoue
frame = picam2.capture_array()
gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)

# ✅ Bon - Conversion d'abord
frame = picam2.capture_array()
frame = cv2.cvtColor(frame, cv2.COLOR_RGB2BGR)
gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
```

**2. Mauvaise interprétation du confidence**
```python
# ❌ Erreur : penser que plus c'est haut, mieux c'est
if confidence > 80:  # FAUX !

# ✅ Correct : LBPH retourne une distance
# Plus c'est BAS, plus les visages sont similaires
if confidence < 100:  # Seuil recommandé
```

**3. Pas assez d'images d'entraînement**
```python
# ❌ Mauvais - moins de 10 images par personne
# Résultats médiocres

# ✅ Bon - 30+ images avec variété
# Meilleure généralisation
```

### 💡 Astuces

**Seuil de confiance ajustable**
```python
# Strict (moins de faux positifs)
if confidence < 50:
    name = names[id_pred]

# Souple (moins de faux négatifs)
if confidence < 100:
    name = names[id_pred]
```

**Affichage du niveau de confiance**
```python
cv2.putText(frame, f'{name} ({confidence:.0f})', (x, y-10),
           cv2.FONT_HERSHEY_SIMPLEX, 0.8, color, 2)
```

**Chargement du modèle sauvegardé**
```python
recognizer = cv2.face.LBPHFaceRecognizer_create()
recognizer.read('trainer.yml')  # Charger modèle existant
```
