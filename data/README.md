# Datasets et Images

Ce dossier contient les données pour les exercices.

**⚠️ Ne pas commit de fichiers lourds (>10MB) sur GitHub !**

## Organisation
- `samples/` - Images d'exemple (légères)
- `test_images/` - Images de test personnelles
- `datasets/` - Liens vers datasets externes (pas les fichiers)

## .gitignore
Les fichiers suivants sont ignorés par git:
- *.jpg, *.png de grande taille
- Vidéos (*.mp4, *.avi)
- Datasets complets
