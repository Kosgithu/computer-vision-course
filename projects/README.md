# Projets

Dossiers pour les projets concrets:
- `detection-visage/` - Détecteur de visages
- `compteur-objets/` - Comptage d'objets en mouvement
- `reconnaissance-chiffres/` - OCR simple
- `projet-perso/` - Ton idée !

Chaque projet doit avoir:
- Un README.md explicatif
- Un dossier `src/` pour le code
- Un fichier `requirements.txt`
