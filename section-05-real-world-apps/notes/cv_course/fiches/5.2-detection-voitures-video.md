# 5.2 - Détection de Voitures dans une Vidéo

## 🎯 Concept

La détection de voitures en temps réel dans une vidéo utilise le même principe que la détection de visages, mais appliqué à un flux vidéo continu. On utilise un classificateur Haar Cascade entraîné spécifiquement pour reconnaître les voitures.

### Points clés
- **Traitement vidéo frame par frame** : Chaque image de la vidéo est analysée individuellement
- **Haar Cascade spécifique** : `haarcascade_car.xml` est entraîné sur des milliers d'images de voitures
- **Contrôle de vitesse** : `time.sleep()` permet de ralentir la lecture pour une meilleure visualisation

---

## 💻 Code Minimal

```python
import cv2
import time

# Chargement du classificateur
cars_detect = cv2.CascadeClassifier('haarcascade_car.xml')

# Ouverture de la vidéo
capture = cv2.VideoCapture('cars.avi')

while capture.isOpened():
    time.sleep(0.05)  # Ralentir la lecture
    
    ret, frame = capture.read()
    gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
    
    # Détection
    cars = cars_detect.detectMultiScale(gray, 1.4, 2)
    
    # Dessin des rectangles
    for (x, y, w, h) in cars:
        cv2.rectangle(frame, (x, y), (x+w, y+h), (255, 0, 0), 3)
    
    cv2.imshow('Voitures détectées', frame)
    
    if cv2.waitKey(1) == 27:
        break

capture.release()
cv2.destroyAllWindows()
```

---

## 🔧 Fonction détaillée

### `cv2.VideoCapture()`
```python
capture = cv2.VideoCapture('cars.avi')  # Fichier vidéo
capture = cv2.VideoCapture(0)            # Caméra en direct
```
- **Paramètre** : Chemin du fichier vidéo OU index de la caméra
- **Retour** : Objet VideoCapture

### `capture.isOpened()`
Vérifie si la vidéo/caméra a été ouverte avec succès.
- **Retour** : `True` si ouvert, `False` sinon

### `time.sleep(seconds)`
```python
time.sleep(0.05)  # Pause de 50ms entre chaque frame
```
- **Utilité** : Ralentir la lecture vidéo pour meilleure visualisation
- **Important** : Sans ça, la vidéo défile trop vite sur les machines rapides

### Paramètres de détection pour voitures
```python
cars_detect.detectMultiScale(gray, 1.4, 2)
```
- **scaleFactor = 1.4** : Plus élevé que pour les visages (1.3) car les voitures ont des tailles très variables
- **minNeighbors = 2** : Plus permissif que pour les visages (5) car les voitures peuvent être partiellement cachées

## 🎨 Analogie Simple

Imagine un vigile devant un écran de surveillance :
- Il regarde **chaque image** (frame) de la vidéo
- Il a un **manuel** (Haar Cascade) avec des photos de voitures
- Il compare chaque image avec son manuel
- Il **entoure** les voitures qu'il repère avec un marqueur
- Il peut **ralentir** la vidéo s'il va trop vite

## ⚠️ Pièges et Astuces

### ❌ Erreurs fréquentes

**1. Vidéo non trouvée**
```python
# ❌ Mauvais
if not capture.isOpened():
    print("Erreur : vidéo inaccessible")
    # Le programme continue quand même !

# ✅ Bon
if not capture.isOpened():
    raise IOError("Impossible d'ouvrir la vidéo cars.avi")
```

**2. Oublier time.sleep()**
```python
# Sans time.sleep(), la vidéo défile trop vite pour être vue
# Sur un PC rapide, une vidéo de 10s peut se lire en 2s !
```

**3. Ne pas vérifier ret**
```python
# ❌ Risqué
ret, frame = capture.read()
gray = cv2.cvtColor(frame, ...)  # Erreur si ret=False

# ✅ Sécurisé
ret, frame = capture.read()
if not ret:
    break
```

### 💡 Astuces

**Ajuster la vitesse de lecture**
```python
time.sleep(0.1)   # Lent (10 fps)
time.sleep(0.05)  # Normal (20 fps)
time.sleep(0.01)  # Rapide (100 fps)
```

**Améliorer la précision**
```python
# Pour moins de faux positifs
cars = cars_detect.detectMultiScale(gray, 1.2, 5)

# Pour détecter plus de voitures (mais plus de bruit)
cars = cars_detect.detectMultiScale(gray, 1.5, 1)
```

**Convertir une vidéo au bon format**
```bash
# Si ta vidéo n'est pas en .avi
ffmpeg -i video.mp4 -c:v mjpeg -q:v 3 cars.avi
```

## 🔗 Différences avec la détection de visages

| Aspect | Visages | Voitures |
|--------|---------|----------|
| **scaleFactor** | 1.3 | 1.4 (plus de variation de taille) |
| **minNeighbors** | 5 | 2 (moins strict, voitures partielles) |
| **time.sleep()** | Non nécessaire | Essentiel (vidéo prédéfinie) |
| **Couleur ROI** | Rouge | Bleu (convention) |
| **Objet** | Visage entier | Voiture complète |

---

## 📝 Résumé

La détection de voitures dans une vidéo combine :
1. **Lecture vidéo** avec `VideoCapture`
2. **Traitement frame par frame** en boucle
3. **Détection Haar Cascade** optimisée pour les véhicules
4. **Contrôle de vitesse** avec `time.sleep()`
5. **Affichage temps réel** avec rectangles colorés

**Point clé** : `time.sleep()` est crucial pour une lecture confortable, et les paramètres de détection sont plus permissifs (1.4, 2) que pour les visages car les voitures ont des formes plus variées.
